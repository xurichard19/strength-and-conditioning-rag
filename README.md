# Strength & Conditioning RAG-based Assistant

A Retrieval-Augmented Generation (RAG)-based Strength & Conditioning (S&C) assistant designed to deliver evidence-based, personalized training insights by combining sports physiology literature with user-specific training logs. The system ingests and processes heterogeneous data sources (PDF research papers, structured workout logs), performs semantic chunking and embedding using Sentence Transformers, and stores vectors in a local Chroma vector database for efficient similarity search. At query time, user inputs are embedded and used to retrieve the most relevant context from both scientific sources and historical performance data, which is then passed to a locally hosted LLM via Ollama for grounded, context-aware response generation. The pipeline emphasizes modular design, incorporating components for document ingestion, text preprocessing, vector indexing, and retrieval optimization, enabling scalable experimentation with techniques such as reranking, contradiction detection, and personalized analytics.
